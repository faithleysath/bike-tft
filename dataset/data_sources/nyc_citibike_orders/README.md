# NYC Citi Bike Orders

## 数据源

- 数据集名称：Citi Bike NYC
- Kaggle 地址：https://www.kaggle.com/datasets/leonczarlinski/citi-bike-nyc
- Kaggle 引用：`leonczarlinski/citi-bike-nyc`

## 下载

```bash
uv run python dataset/data_sources/nyc_citibike_orders/download_kaggle_dataset.py leonczarlinski/citi-bike-nyc
```

默认下载并解压到：

```text
dataset/data_sources/nyc_citibike_orders/raw/citi-bike-nyc/
```
