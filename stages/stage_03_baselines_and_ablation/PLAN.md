# Stage 03 Plan

## 阶段目标

把 AGCRN 训练成后续阶段的主模型，并在此基础上构建完整的实验比较体系，回答下面三个问题：

1. AGCRN 能否在增强版站点级数据上稳定跑通，并成为比阶段 1 TFT 更强的主模型
2. AGCRN 相比 TFT 和常见基线是否更好
3. AGCRN 里的哪些模型机制真正有效

## 计划任务

- [ ] 明确 AGCRN 输入张量设计，包括站点静态特征、动态特征和外生变量
- [ ] 设计站点图构造方式，至少确定一版初始邻接关系或自适应图方案
- [ ] 跑通 AGCRN 数据准备、训练、验证和推理流程
- [ ] 固定一版代表性的 AGCRN run
- [ ] 保存 checkpoint、日志、配置和报告
- [ ] 确定基线模型列表
- [ ] 统一训练 / 验证 / 测试切分
- [ ] 统一指标口径
- [ ] 跑通至少一组传统或机器学习基线
- [ ] 与阶段 1 的 TFT 基线和阶段 2 的增强版数据实验做对比
- [ ] 做 AGCRN 的模型级消融，而不是字段级消融
- [ ] 形成汇总表和结论

## 本阶段输出位置约定

- 原始数据：`data/raw/stage_03_baselines_and_ablation/`
- 处理结果：`data/processed/stage_03_baselines_and_ablation/`
- 训练结果：`runs/stage_03_baselines_and_ablation/`

## 完成标准

- 至少有一版可复现的 AGCRN 代表性 run
- 有统一的对比实验表
- 有清晰的 AGCRN 模型级消融结论
- 结果能直接写入论文实验章节
