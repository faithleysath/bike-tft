# 图片素材

当前 `v1` 演示文稿未使用外部图片占位符，所有视觉内容都由内联 CSS / SVG 构成，因此可以直接打开 HTML 使用。

如果后续想替换为真实图表截图或实验图片，可以把素材放入本目录，并在下一版中按需要接入。

建议命名规则：

| 序号 | 用途 | 建议分辨率 | 格式 |
| --- | --- | --- | --- |
| 01 | 模型结果图 | 1600×900 | PNG / JPG |
| 02 | 解释性图表 | 1600×900 | PNG / JPG |
| 03 | 校园迁移示意图 | 1600×900 | PNG / JPG |

命名示例：`01.png`、`02.jpg`
