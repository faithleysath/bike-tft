# Graph WaveNet Foreign Translation Source

This directory stores source materials for the graduation thesis foreign-language translation task.

## Paper

- Title: Graph WaveNet for Deep Spatial-Temporal Graph Modeling
- Authors: Zonghan Wu, Shirui Pan, Guodong Long, Jing Jiang, Chengqi Zhang
- Venue: Proceedings of IJCAI 2019, pages 1907-1913
- DOI: https://doi.org/10.24963/ijcai.2019/264
- arXiv: https://arxiv.org/abs/1906.00121
- Download date: 2026-04-29

## Files

- `graph_wavenet_ijcai2019.pdf`: official IJCAI proceedings PDF.
- `graph_wavenet_ijcai2019.bib`: BibTeX downloaded from the IJCAI proceedings page.
- `arxiv_1906.00121_source.tar.gz`: arXiv TeX source package.
- `latex_source/`: extracted arXiv source package.

## LaTeX Entry Points

- `latex_source/ijcai19-graph-wavenet.tex`: top-level IJCAI LaTeX wrapper.
- `latex_source/main.tex`: main paper body, useful for translation.
- `latex_source/fig/`: original figure PDFs.
